# PRIM-USLUGI Service Platform

Premium UI/UX Service Platform built with Next.js, Framer Motion, and Tailwind CSS.

## Features
- 🚀 Modern Premium Design (Glassmorphism)
- 📱 Fully Responsive (Side-Drawer Mobile Navigation)
- ⚡ High Performance (Next.js 14)
- 🎨 Real-time Background Dynamic Interaction
- 📞 Telegram Bot Integration (SMTP/Polling)

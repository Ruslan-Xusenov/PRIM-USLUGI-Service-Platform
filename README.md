# 📦 Платформа Грузоперевозок "Прим-Услуги" (PRIM-USLUGI)

Добро пожаловать в репозиторий платформы **"Прим-Услуги"** — современного и высоконагруженного веб-приложения для предоставления логистических и транспортных услуг. Платформа предназначена для удобного заказа грузчиков, комиссаров и грузового транспорта.

## 🎯 Цель проекта

Основная цель проекта — автоматизировать и упростить процесс заказа различных логистических услуг как для клиентов, так и для администраторов. 
Платформа оснащена:
- **Пользовательским интерфейсом** (UI) для комфортного ознакомления с услугами и оформления заявок.
- **Интеграцией с Telegram API**, который моментально уведомляет администратора о новых заказах, заявках с сайта и сообщениях клиентов.
- **Мощной Административной Панелью**, в которой можно управлять новостями, создавать полностью новые (SEO-оптимизированные) страницы и управлять собственной медиатекой.

## 🚀 Технологии и Стек

Проект разработан с использованием самых современных и стабильных технологий:
- **Next.js 14+ (App Router)** — Основной полностековый фреймворк для Server-Side Rendering (SSR) и создания API-обработчиков.
- **React.js** — Библиотека для создания динамичного пользовательского интерфейса.
- **Node.js** — Среда выполнения JavaScript на сервере.
- **SQLite** — Легковесная и быстрая локальная база данных, используемая для хранения динамических страниц, новостей и медиа-ссылок.
- **CSS / Vanilla CSS** — Для премиального визуального оформления, в том числе полностью отзывчивого (Mobile-First) дизайна административной панели без зависимостей от сторонних компиляторов.
- **Framer Motion** — Для создания плавных анимаций и графических переходов на страницах сайта (интерактивы).
- **Telegraf / Node-Telegram-Bot-API** — Для связи сайта с экосистемой Telegram (моментальная отправка заявок администраторам).
- **React-Quill / Quill.js** — Полноценный визуальный редактор текста (Rich Text Editor) для создания страниц и новостей прямо из админ-панели (без программирования).

## ✨ Основные возможности (Функционал)

1. **Оформление услуг**: Отдельные адаптивные страницы для Грузчиков, Аварийных Комиссаров, Вывоза мусора и Грузоперевозок с уникальными баннерами.
2. **Админ-Дашборд**: Удобная панель управления (Responsive Admin Dashboard), стилизованная градиентами, эффектами стекла (`glassmorphism`) и карточным дизайном для мобильных устройств.
3. **Медиатека (Media Library)**: Прямая загрузка изображений на сервер и простое копирование ссылок (URL) нажатием одной кнопки для добавления картинок на сайт.
4. **Конструктор Страниц (CMS)**: Создание абсолютно новых динамических страниц сайта (например, `/about`, `/contacts`) с помощью встроенного визуального редактора, включая точную настройку SEO (Title, Description, Keywords).
5. **Система Новостной Ленты**: Публикация "Акций", "Обновлений" и "Новостей" на главной странице платформы для удержания внимания клиентов.
6. **Telegram Оповещения**: Формы обратной связи строго валидируются и мгновенно отправляют структурированные и красивые сообщения в Telegram.

## 🛠 Установка и Запуск (Для разработчиков)

Для развертывания проекта на локальном или продакшн сервере следуйте инструкции:

### 1. Клонирование репозитория
```bash
git clone https://github.com/Ruslan-Xusenov/PRIM-USLUGI-Service-Platform.git
cd PRIM-USLUGI-Service-Platform
```

### 2. Установка зависимостей
Убедитесь, что у вас установлен Node.js версии не ниже 18.x.
```bash
npm install
```

### 3. Настройка переменных окружения
Создайте файл `.env.local` в корне проекта (или скопируйте `.env.example`, если есть) и пропишите туда необходимые данные для подключения Telegram-бота:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
ADMIN_CHAT_ID=ваш_чат_айди
```

### 4. Запуск проекта в режиме разработки
Проект настроен на одновременный запуск Next.js клиента и фонового процесса бота (прослушка команд) с помощью утилиты `concurrently`, которая определена в скрипте `runner.js`.
```bash
npm run dev
# или альтернативный запуск напрямую:
node runner.js
```

После этого:
- Пользовательская часть и сайт будут доступны по адресу: `http://localhost:3000`
- Телеграм-бот активируется и начнет обрабатывать входящие заявки.

## 📂 Структура проекта

* `src/app/` — Маршрутизация и основные страницы сайта (Next.js App Router).
* `src/app/admin/` — Защищенная зона административной панели.
* `src/app/api/` — Серверные эндпоинты (Backend API Next.js).
* `src/components/` — Переиспользуемые React-компоненты (AdminLayout, PageEditor, Navigation).
* `src/lib/` — Служебные скрипты (инициализация базы данных SQLite, функции для работы с API, логика бота).
* `public/` — Статические файлы (изображения, иконки).
